Basic Customization

GENERAL

Concierge template is a set of responsive multi-purpose pages with multiple layout versions. This documentation will help you understand and customize the template for your needs.



CHANGING COLOR SKIN

Concierge offers two skins to choose from. In order to change the default one, you need to add a child css file link inside the HEAD tag, like:

<link rel="stylesheet" type="text/css" href="css/gold.css">

Remember that it needs to be placed after a default css file link, which looks like:

<link rel="stylesheet" type="text/css" href="css/style.css">

You will need to do this for all HTML files, but you can use something like "mass replace in files" function of your text editor ("Search / Find in Files..." if you are using Notepad++) to make it quick.



REMOVING STYLE SWITCHER

Style switcher is used to demonstrate all color skins on demo page, so most probably you don't want to have it on your live site. To do this, you need to open the js/script.js file, scroll down to STYLE SWITCHER section and delete all the code inside it.



CHANGING PAGE HEADER

This template comes with two header variations. Default one is 'triangle'. The variation can be choosed by editing class of the HEADER tag in each HTML file. In order to use 'triangle':

<header id="header" class="triangle">


If you want to choose a 'classic' version:

<header id="header" class="classic">

Maps

We use the GMap3 jQuery plugin for handling Google Maps easily. The maps are visible for example, on Homepage 4 and Locations. Using them is very easy, but each map needs to be configured separately.

Every map is defined in the similar way in js/script.js file, for example:

$('#map-our_location').width("100%").height("468px").gmap3({
	map:{
		options: gmapOptions,
	},
	styledmaptype: subtleOptions,
	marker:{
		values: [
			{
				latLng:[44.28952958093682, 6.152559438984804],
				options:{
					icon: "img/marker.png"
				}
			}
		],
	}
},"autofit");

You can define the width and height of the map and specify the marker points. If you want to add overlays, custom events or even more, take a look on this plugin documentation here: http://gmap3.net/

Fleet filtering

To display and filter the fleet contained in second tab on the Homepage 4 we use an Isotope jQuery plugin.

Here is the example code where you declare the filters

<ul class="custom-list fleet-filters clearfix">
	<li><a href="#" class="btn dark active" data-filter="*">All</a></li>
	<li><a href="#" class="btn dark" data-filter=".yachts">Yachts</a></li>
	<li><a href="#" class="btn dark" data-filter=".jets">Jet Planes</a></li>
	<li><a href="#" class="btn dark" data-filter=".luxcars">Luxury Cars</a></li>
	<li><a href="#" class="btn dark" data-filter=".cars">Cars</a></li>
</ul>

As you can see, you have to use data-filter parameter in order to select specified classes of items. Each item, defined below this code, has a proper class that makes possible to sort them:

<li class="luxcars">
 [...]
</li>
<li class="jets">
 [...]
</li>
<li class="cars">
 [...]
</li>

If you want to customize Isotope filtering even more, take a look on this plugin documentation here: http://isotope.metafizzy.co/
